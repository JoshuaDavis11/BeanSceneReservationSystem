# BeanScene Reservation System

A full-stack reservation and event management system designed for a caf√© or venue environment. The system supports multiple user roles and provides staff and managers with tools to manage reservations, seating, and events efficiently.

## Overview

This project was built to simulate a real-world reservation workflow, from customer bookings through to staff operations and management oversight. It focuses on role-based access, data management, and practical business functionality.

## Key Features

**User Accounts**
- Create accounts and make reservations
- View and manage personal bookings

**Staff Functionality**
- View and manage customer reservations
- Assign seating arrangements
- Support day-to-day reservation operations

**Manager Functionality**
- Create and manage events
- Manage user accounts
- Oversee system usage and reservations

**Role-Based Access Control**
- Separate permissions for users, staff, and managers
- Secure access to features based on role

## Technical Highlights

- Full-stack application with frontend and backend components
- RESTful API for handling reservations, users, and events
- Role-based access control and structured data flow
- Designed to reflect real business workflows

## Technologies Used

- C# and .NET
- JavaScript
- HTML and CSS
- RESTful APIs
- SQL (database-backed storage)
- Git and GitHub
